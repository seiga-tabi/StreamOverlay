당신은 Twitch 롤 시참방송 자동화 시스템 설계자입니다.

현재 participation.module을 개선하세요.

목표:
1. !시참 RiotID#태그 신청 안정화
2. !참가확인 처리
3. !다음시참 처리
4. 대기열 overlay 업데이트
5. 대시보드에서 대기열이 보기 좋게 표시되도록 API/상태 개선
6. Riot API가 설정된 경우 PUUID 조회, 없으면 형식 검증만 수행
7. 동일 Twitch 유저와 동일 Riot ID 중복 신청 방지
8. 노쇼 처리 helper 추가

안전 규칙:
- Riot ID는 OBS overlay에 기본 표시하지 마세요.
- viewer input을 shell/file/url로 연결하지 마세요.
- 모든 상태 변경은 로그로 남기세요.

검증:
- npm run build
- npm run validate:config
