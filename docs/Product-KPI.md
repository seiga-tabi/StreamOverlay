# YORO.gg Product KPI v1

문서 상태: 공식 제품 KPI 기준  
적용 범위: 제품 성과 측정, 로드맵 우선순위, 실험 평가

## 1. North Star KPI

YORO.gg의 North Star KPI는 “Streamer와 Viewer가 연결된 의미 있는 참여 이벤트 수”이다.

포함 이벤트:

- 팔로우 스트리머 전적 확인
- 시참 신청
- 시참 참여 완료
- 스트리머 커뮤니티 참여
- 대회 참여 또는 공유
- overlay를 통한 방송 참여 반응

## 2. Guest KPI

- 검색 성공률
- 검색 결과 로딩 성공률
- Riot ID 검색 후 profile view rate
- profile에서 streamer discovery click rate
- Guest to Twitch login conversion
- 즐겨찾기 저장률
- 첫 방문 후 7일 재방문율

## 3. User KPI

- Twitch login completion rate
- Followed streamer view rate
- Participation join rate
- Participation completion rate
- Queue cancel rate
- Community post/comment rate
- Favorite reuse rate
- 7일 / 30일 retention

## 4. Streamer KPI

- Streamer registration completion rate
- Riot ID approval completion time
- Dashboard weekly active streamers
- Broadcast readiness success rate
- 방송 준비 평균 시간
- Participation open rate
- Participation queue completion rate
- Overlay source setup success rate
- Overlay usage rate
- Streamer 30일 retention

## 5. Admin KPI

- Approval 처리 시간
- Twitch / EventSub error detection time
- Incident recovery time
- Failed action rate
- Overlay disconnected session rate
- API key configuration success rate
- Admin task completion time

## 6. Overlay KPI

- OBS 연결 성공률
- Source URL copy rate
- Preview success rate
- Test event success rate
- Live overlay connected minutes
- Overlay reconnect count
- Source별 사용률

## 7. Community KPI

- Streamer Space visit rate
- Party Finder post creation rate
- Party fulfillment rate
- Server / Clan recruitment conversion
- Comment participation rate
- Community returning user rate
- Streamer community follower growth

## 8. Tournament KPI

- Tournament creation completion rate
- Team registration completion rate
- Public tournament page share rate
- Bracket view rate
- Schedule view rate
- Tournament participant retention

## 9. Quality KPI

- Core Web Vitals
- Error rate
- API failure rate
- Mobile task completion rate
- Accessibility issue count
- Design System component adoption rate
- Deprecated component usage count

## 10. KPI 사용 원칙

- KPI는 기능 추가의 근거가 아니라 사용자 가치 검증 도구이다.
- 단순 pageview는 보조 지표이다.
- North Star에 연결되지 않는 KPI는 우선순위 판단에 사용하지 않는다.
- Streamer와 Viewer 연결을 증가시키는 지표를 우선한다.
