# YORO.gg Community Philosophy v1

문서 상태: 공식 Community 제품 철학  
적용 범위: Community, Party Finder, Server Recruitment, Streamer Space, Tournament Community

## 1. Community 정의

YORO.gg Community는 게시판이 아니다.

YORO.gg Community는 Streamer Community이다.

사용자는 글을 보러 오는 것이 아니라 함께 플레이할 사람, 함께 볼 방송, 참여할 커뮤니티를 찾으러 온다.

## 2. Community의 목적

Community의 목적은 다음이다.

- 시청자가 스트리머와 방송 밖에서도 연결되게 한다.
- 시참 경험을 일회성 queue가 아니라 관계로 확장한다.
- 파티, 서버, 대회, 공지를 스트리머 중심으로 묶는다.
- Viewer가 팬과 커뮤니티 멤버로 성장하게 한다.

## 3. Community Areas

Community는 다음 영역으로 구성한다.

- Streamer Space
- Live Participation
- Party Finder
- Server / Clan Recruitment
- Tournament Hub
- Broadcast Announcements
- Viewer Reputation

## 4. Streamer Space

Streamer Space는 스트리머별 커뮤니티 홈이다.

포함 정보:

- 방송 상태
- Riot profile
- 참여 가능 여부
- 공지
- 커뮤니티 글
- 대회
- 최근 참여자
- 팔로워 activity

## 5. Party Finder

Party Finder는 일반 게시판이 아니라 즉시 함께 플레이할 사람을 찾는 기능이다.

필수 정보:

- Riot ID
- Rank
- Role
- Mode
- Voice 가능 여부
- 모집 인원
- 만료 시간

## 6. Server / Clan Recruitment

Server / Clan Recruitment는 장기 관계를 위한 공간이다.

필수 정보:

- 커뮤니티 성격
- 주 활동 시간
- 주 언어
- 참여 조건
- 스트리머 연결 여부
- Riot profile preview

## 7. Viewer Reputation

시참과 커뮤니티는 신뢰가 필요하다.

YORO.gg는 다음 신뢰 신호를 제품적으로 다룬다.

- 참여 이력
- no-show 여부
- 선호 role
- 최근 activity
- 커뮤니티 기여

신뢰 신호는 처벌보다 좋은 매칭과 안전한 운영을 위해 사용한다.

## 8. Community 성공 기준

- 게시글 목록보다 참여 가능한 관계가 먼저 보인다.
- 스트리머별 커뮤니티 맥락이 유지된다.
- 시청자가 방송 밖에서도 다시 돌아올 이유가 있다.
- 커뮤니티 활동이 스트리머 성장으로 연결된다.
