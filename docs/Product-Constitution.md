# YORO.gg Product Constitution v1

문서 상태: 공식 제품 기준  
적용 범위: YORO.gg 전체 제품, 디자인, 개발, 운영, 커뮤니티 정책  
최상위 원칙: YORO.gg는 전적검색 사이트가 아니라, 스트리머와 시청자가 함께 게임을 즐기는 플랫폼이다.

## 1. 제품 존재 이유

YORO.gg는 League of Legends를 중심으로 스트리머와 시청자가 함께 플레이하고, 발견되고, 연결되고, 성장하는 경험을 만들기 위해 존재한다.

일반 전적검색 서비스는 게임 데이터를 보여준다. YORO.gg는 그 데이터를 관계와 참여로 연결한다. 사용자는 단순히 누가 잘하는지 확인하는 데서 끝나지 않고, 그 플레이어가 방송 중인지, 함께 참여할 수 있는지, 어떤 커뮤니티를 운영하는지, 어떤 대회를 여는지까지 자연스럽게 이어져야 한다.

YORO.gg의 핵심 존재 이유는 다음이다.

- 시청자가 좋아할 스트리머를 발견하게 한다.
- 스트리머가 오늘 방송을 안정적으로 운영하게 한다.
- 전적, 참여, 커뮤니티, 대회를 하나의 흐름으로 연결한다.
- KR / JP League of Legends 방송 문화를 더 쉽게 참여 가능한 경험으로 만든다.

## 2. 제품 철학

YORO.gg는 검색보다 관계를 우선한다.

검색은 시작점이다. 목표는 검색 결과 안에서 사람, 방송, 참여, 커뮤니티를 발견하게 하는 것이다. 모든 기능은 “이 기능이 스트리머와 시청자를 더 가깝게 만드는가?”라는 질문을 통과해야 한다.

YORO.gg는 기능의 개수보다 흐름의 명확성을 우선한다.

스트리머에게 많은 설정을 제공하는 것보다, 오늘 방송을 성공시키는 데 필요한 일을 빠르게 끝내는 것이 더 중요하다. 시청자에게 많은 메뉴를 보여주는 것보다, 지금 참여 가능한 방송과 커뮤니티를 쉽게 찾게 하는 것이 더 중요하다.

YORO.gg는 방송 안정성을 제품 품질의 일부로 본다.

방송 중 사용되는 기능은 실패했을 때의 영향이 크다. 따라서 모든 방송 관련 기능은 안전하고 예측 가능하며 되돌릴 수 있어야 한다.

## 3. 만드는 기능

YORO.gg는 다음 기능을 만든다.

- League of Legends Riot ID 검색과 전적 분석
- 스트리머 발견, 프로필, 방송 상태, Riot ID 연결
- Twitch 로그인 기반 시청자 경험
- 시청자 참여 신청, 대기열, 체크인, 초대, 노쇼 관리
- OBS Browser Source overlay 운영
- 방송 중 알림, 채팅, 자막, 시참 대기열, 솔로랭크 정보 표시
- 스트리머의 오늘 방송 준비와 운영 dashboard
- 스트리머 중심 커뮤니티, 파티 모집, 서버/클랜 모집
- 스트리머 대회 운영과 공개 대회 경험
- Admin 승인, 안전 설정, 시스템 상태 확인

## 4. 만들지 않는 기능

YORO.gg는 다음 방향의 기능을 만들지 않는다.

- 단순 조회 수를 늘리기 위한 무관한 콘텐츠 피드
- 스트리머와 시청자 연결에 기여하지 않는 일반 게시판 확장
- 방송 중 안전성을 해칠 수 있는 임의 명령 실행 기능
- 시청자 입력으로 arbitrary file write, shell command, arbitrary URL open, OBS 임의 제어가 가능한 기능
- 운영자 승인 없이 위험한 moderation, stream start/stop, stream key 변경을 수행하는 기능
- 기존 flow로 해결 가능한 문제를 새 페이지 추가로 해결하는 기능
- Design System 없이 화면별로 독립 스타일을 추가하는 기능
- KR / JP 사용자를 고려하지 않은 단일 언어 중심 기능

## 5. 장기 목표

YORO.gg의 장기 목표는 KR / JP League of Legends 스트리머와 시청자가 방송 참여, 커뮤니티, 대회를 운영하는 표준 플랫폼이 되는 것이다.

5년 목표:

- 전적검색에서 스트리머 발견으로 이어지는 대표 서비스가 된다.
- 스트리머가 OBS overlay와 시참 운영을 1분 안에 준비할 수 있다.
- 시청자는 팔로우 중인 스트리머의 시참, 대회, 커뮤니티를 한곳에서 확인한다.

10년 목표:

- YORO.gg는 KR / JP 게임 방송 커뮤니티 운영 인프라가 된다.
- League of Legends를 시작점으로, 방송 참여형 게임 경험을 플랫폼화한다.
- 스트리머 성장, 시청자 참여, 커뮤니티 활동, 대회 운영이 하나의 데이터 loop로 연결된다.

## 6. 성공 기준

YORO.gg는 다음 기준을 만족할 때 성공한 제품이다.

- 사용자가 3초 안에 Riot ID를 검색할 수 있음을 이해한다.
- 사용자가 30초 안에 스트리머, 방송 상태, 참여 가능 여부를 이해한다.
- 스트리머가 1분 안에 오늘 방송 가능 상태를 판단한다.
- OBS overlay 설정이 3클릭 안에 가능하다.
- 시청자 참여 신청부터 방송 참여까지 흐름이 끊기지 않는다.
- Public, My YORO, Streamer Studio, Overlay Studio, Admin이 같은 제품처럼 보이고 작동한다.
- 새 기능이 추가되어도 Design System과 Product Principles가 깨지지 않는다.
